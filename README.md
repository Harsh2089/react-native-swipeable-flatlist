<h1>React Native Swipeable FlatList</h1>



You can clone the repository and use the following commands:

##### Install EXPO SDK, if you don't have
```shell
npm install -g expo-cli
```

##### Install dependancies
```shell
npm install
```

##### Run code
```shell
expo-cli start
```
